## 1. 증권성 판단 — 3시나리오 대응안

### 배경

자본시장법 §4에 따라 금융투자상품의 "증권성" 판단은 STO 발행 구조 전체를 결정한다.  
NPL 회수권 토큰은 아직 **국내 선례가 없는 상품 유형**이다. 아래 3가지 시나리오 중 어느 경로로  
규제당국이 판단하느냐에 따라 SPC 설계·발행 절차·투자자 모집 방식이 달라진다.

> 📌 어느 시나리오로 귀결될지는 **법무법인 legal opinion으로 확정**해야 한다.  
> 아래는 각 시나리오 발생 시 사업 구조에 미치는 영향을 정리한 내부 대응안이다.

---

### 시나리오 A — 투자계약증권으로 인정 (자본시장법 §4 ⑥)

**판단 근거 가능성**: NPL 회수권을 "타인의 노력(AMC 서비서의 채권 회수 활동)에 의한 손익"을 공유하는 계약으로 볼 경우 투자계약증권 해당 가능성 있음.

| 구분 | 내용 |
|------|------|
| 공모 여부 | 50인 이상 → 공모. 증권신고서 제출 의무 발생 (금융위 심사 ~45 영업일) |
| 사모 트레이드오프 | 50인 미만 사모 구성 가능. 단, 유통 제한·투자자 수 상한으로 초기 규모 제약 |
| SPC 역할 | 투자계약증권 발행인 = SPC. 발행인 요건 충족 필요 |
| 유통 채널 | 2026.1 개정으로 증권사 유통 허용 — 단, 증권신고서 효력 발생 후 가능 |
| 투자자 보호 | 일반투자자 투자한도 규제 적용 (자본시장법 시행령 기준) |

**사업 구조 변화**:
- 공모 경로: 발행 준비 기간 +45~60 영업일 (증권신고서 심사). 첫 발행 일정이 후방 이동.
- 사모 경로(50인 미만): 규제 부담은 낮으나 초기 자금 조달 상한이 생김. LP 유치 시 사모펀드 구조와 결합 필요.
- 양 경로 모두: **법무법인 의견서 + 금감원 사전 컨설팅 必須** → T2 컨펌 후 진행.

**샌드박스 활용 포인트**: 공모 증권신고서 심사 기간을 단축하거나, 일반투자자 한도 특례를 받는 방향으로 신청 가능.

---

### 시나리오 B — 수익증권 (신탁 기반, 자본시장법 §4 ④)

**판단 근거 가능성**: NPL 채권을 신탁회사(신탁업자)에 신탁하고, 그 수익권을 토큰으로 발행하는 구조로 설계할 경우 수익증권에 해당. 구조 변경이 필요하지만 상대적으로 선례가 풍부함.

| 구분 | 내용 |
|------|------|
| 구조 전제 | SPC 직접 발행 → **신탁회사 수탁 + SPC는 위탁자** 구조로 전환 필요 |
| 발행인 | 신탁회사 (신탁업자 등록 기관). SPC는 위탁자·수익자 지정 역할 |
| SPC 설계 변경점 | `spc_vehicle.trust_type = 'trust'` 선택. 신탁계약서 작성 + 신탁업자 제휴 필수 |
| 공모/사모 기준 | 동일하게 50인 기준 적용 |
| 장점 | 신탁 구조는 기존 법제(신탁법) 안에서 상대적으로 명확. 법적 안정성이 A보다 높음 |
| 단점 | 신탁업자 선정·계약·수수료 구조 추가 협상 필요. 신탁보수로 IRR 소폭 감소 가능 |

**사업 구조 변화**:
- `spc_vehicle` 테이블의 `trust_type` 필드를 `trust`로 설정 + 신탁계약 참조 필드 추가.
- 신탁회사 선정 후 업무위탁협약(MOU) 필요 → T2 컨펌 후 진행.
- AMC 서비서(채권 회수 대리인)는 신탁회사와 별도 위탁 계약으로 연결.

**샌드박스 활용 포인트**: "NPL 신탁 수익권 토큰화"로서 신탁업 인가 요건 완화 특례 또는 신탁업자 제휴 간소화 특례 신청 가능.

---

### 시나리오 C — 증권성 불인정

**판단 근거 가능성**: NPL 회수권이 단순 채권 양수도(채권 매매)에 해당하며 "집합투자" 또는 "수익 공유" 구조가 아니라고 판단되는 경우. 가능성은 낮으나 완전 배제 불가.

| 구분 | 내용 |
|------|------|
| 리스크 | 증권 규제 밖에 놓이므로 제도권 투자자 보호 장치 미비. 법적 공백 지대 |
| 유통 경로 | 증권사 유통 불가. 자체 플랫폼으로만 거래 → 유동성 확보 어려움 |
| 투자자 신뢰 | 사모펀드·연기금·HNWI LP 유치 시 "증권이 아닌 것"의 법적 성격이 불명확하다는 이유로 거부 가능성 |

**대안 경로 (증권성 불인정 시)**:
1. **샌드박스 특례로 한시적 허용**: "혁신금융서비스"로 지정받아 증권성 판단과 무관하게 2년 운영 특례 획득 → 운영 중 선례·데이터 축적 후 법제화 추진.
2. **구조 재설계**: 수익증권(시나리오 B) 방향으로 신탁 구조를 명시적으로 채택 → 증권성을 스스로 확정.
3. **사모펀드(PEF) 경로**: 4단계 모델 중 GP/펀드 운용 단계를 앞당겨, 투자자와의 관계를 LP 출자 계약으로 구성. STO 없이 운용 → 이후 RWA 토큰화 재도전.

**사업 구조 변화**:
- 시나리오 C는 우리가 원하는 방향이 아님. **적극적으로 시나리오 A 또는 B를 확정짓는 것이 목표**.
- 샌드박스 신청이 시나리오 C 리스크에 대한 가장 빠른 완충 장치.

---

### 3시나리오 구조 영향 비교표

| 항목 | A: 투자계약증권 | B: 수익증권(신탁) | C: 증권성 불인정 |
|------|-----------------|-------------------|-----------------|
| SPC 구조 | SPC 직접 발행 유지 | 신탁회사 수탁 추가 | SPC 불필요 가능 |
| 발행 선행 조건 | 증권신고서 또는 사모 50인 제한 | 신탁업자 제휴 계약 | 없음 (단, 법적 공백) |
| 증권사 유통 | 가능 (신고서 효력 후) | 가능 (수익증권) | 불가 |
| 공모 투자자 모집 | 신고서 후 가능 | 신고서 후 가능 | 불가 |
| IRR 영향 | 중립 | 신탁보수만큼 소폭 감소 | 중립 (단, 유동성 프리미엄 소멸) |
| 법적 안정성 | 선례 없음 → 불확실 | 신탁법 체계 → 상대적 안정 | 최악 (법적 공백) |
| 샌드박스 필요성 | 높음 (공모 선례 확보) | 중간 (신탁 선례 있음) | 필수 (유일한 운영 경로) |
| **권고 전략** | 사모로 시작 후 공모 전환 | **가장 현실적인 1안** | 회피 대상 |

> **내부 전략 방향**: 시나리오 B(수익증권·신탁 구조)를 1안으로 설계하고, 법무법인 의견서에서 A로 귀결될 경우를 대비한 사모 50인 규모 파일럿을 병행 준비한다.  
> 샌드박스는 양 시나리오 공통으로 "선례 확보·공모 특례"를 위해 신청한다.

## 2. 혁신금융서비스 신청 개요

### 2.1 혁신금융서비스 지정 제도 요지

**근거법**: 금융혁신지원 특별법 (2019.4 시행, 이하 "금융혁신법")

| 항목 | 내용 |
|------|------|
| 제도 목적 | 기존 금융법령상 허가·인가·등록이 없거나 명시적 금지로 신규 서비스 출시가 불가한 경우, 한시적 규제 특례를 부여하여 테스트 가능하게 함 |
| 지정 효과 | 지정 기간(최초 2년, 최대 2년 연장) 동안 특례 대상 규제 적용 면제. 금융위·금감원 밀착 모니터링 |
| 신청 기관 | 금융위원회 (금융혁신심사위원회 심의) |
| 심사 기간 | 신청 접수 후 약 30~60 영업일 (복잡도에 따라 상이) |
| 지정 후 | 지정 기간 종료 전 법·시행령 개정 또는 기존 법령 해석으로 정규화 추진 |
| 투자자 보호 요건 | 서비스 한도 설정, 위험고지, 손실 보전 방안 등 위원회 협의 사항 준수 |

### 2.2 본 사업이 혁신금융서비스에 해당하는 이유

**혁신성 근거 (2가지 축)**:

1. **비유동 자산의 유동화** — NPL은 전통적으로 기관 전용 장외 거래 시장(전임사 5곳 95% 독점)에서만 거래되던 자산. 이를 블록체인 기반 STO로 5만 원 단위 조각화하여 일반투자자에게 24/7 접근 가능하게 하는 것은 기존 금융 서비스에 없는 새로운 투자 경로.

2. **데이터 기반 신뢰구간 공개** — 기존 NPL 매각은 "한 점 평가(point estimate)"로 투자자가 불확실성을 모름. 본 사업은 회수 cone(p10~p90)을 투자자에게 공개하여 위험 기반 의사결정 지원. 이는 기존 NPL 관련 금융 서비스에 없는 혁신적 투자자 보호 방식.

**소비자 편익**:
- 개인 투자자: 기관 전용이었던 고수익(IRR 15~25%) 자산 접근 가능
- 금융시장: NPL 비유동성 리스크 분산 → 시스템 리스크 감소
- 금융기관(은행): BIS 비율·IFRS9 부담 자산 빠른 현금화 채널 확보

### 2.4 2026.1 STO 법제화와 샌드박스의 관계

**법제화됐는데 왜 샌드박스가 또 필요한가?**

2026년 1월 전자증권법·자본시장법 개정으로 STO는 제도권에 편입됐다. 그러나 법제화는 "STO라는 형식"을 허용한 것이고, **"NPL 회수권이라는 특수한 내용물"의 증권성 판단은 별개 문제**다.

| 구분 | 2026.1 법제화가 해결한 것 | 여전히 불확실한 것 |
|------|--------------------------|-------------------|
| 형식 | 분산원장 = 법적 전자등록계좌부 인정 | |
| 형식 | 투자계약증권 증권사 유통 허용 | |
| 내용 | | **NPL 회수권의 증권성 판단 선례 부재** |
| 내용 | | 공모 시 증권신고서 심사 기간(~45영업일) |
| 내용 | | 일반투자자 투자한도 구체 기준 |
| 내용 | | NPL 회수 cone 공개 방식의 적정성 |

**샌드박스의 역할**: NPL 회수권 토큰이 투자계약증권/수익증권 어느 것으로 귀결되든, 공모 절차 없이 **일정 규모까지 실거래 선례**를 먼저 만들고, 투자자 보호 방안의 적정성을 규제당국과 협의하며 확정짓는 수단. 법제화 이후에도 샌드박스가 필요한 이유는 **선례 부재 해소 + 공모 심사 기간 단축 특례 확보**.

| **서비스 개요** | 부실채권(NPL) 회수권을 STO(증권형 토큰)로 발행하여, 소액 개인투자자도 IRR 15~25% 수준의 담보부 NPL에 조각 투자 가능하게 하는 플랫폼. 데이터 기반 회수 신뢰구간(p10~p90)을 투자자에게 투명 공개. |
| **보유 파이프라인** | 7.7조원 / 2.5만 건 NPL 리스트 (내부 보유) |
| **발행 구조** | SPC 또는 신탁 구조 → 분산원장 기반 RWA 토큰 발행 → 투자자 조각 지분 |

### 3.2 혁신성 및 소비자 편익

**혁신성**:
- 기관 전용 자산(NPL)의 소액화·조각화로 **일반투자자 접근성** 신규 창출
- AI 기반 NPL 스코어링으로 **"한 점 평가 → 신뢰구간 평가"** 전환 (투자자 위험 가시화)
- 회수 완료 즉시 토큰 소각·분배로 **자본 순환 1.5~2회/년** (기존 NPL 현금화보다 빠름)

**소비자 편익**:
- 최소 [대표 확정 필요 — 최소 청약 단위] 원부터 참여 가능
- 기존 장외 NPL 시장 불투명성 → 회수 cone 공개로 위험 인식 가능
- 분산원장 기록으로 투자 이력·분배 이력 실시간 조회

### 3.3 규제 특례 요청 사항

> 📌 아래는 내부 초안. 실제 신청 조항 번호·특례 내용은 **법무법인 자문 후 확정** [T2].

| # | 요청 특례 | 대상 법령(초안) | 요청 내용 |
|---|-----------|----------------|-----------|
| 1 | NPL 회수권 토큰 증권성 한시 면제 또는 확인 | 자본시장법 §4 | 특례 지정 기간(2년) 중 NPL 회수권 토큰을 특정 투자증권으로 인정하고, 이를 전제로 발행·유통 허용 |
| 2 | 공모 증권신고서 심사 기간 단축 | 자본시장법 §119 | 표준 45영업일 심사를 샌드박스 지정 사업에 한해 단축 또는 약식 처리 허용 |
| 3 | 일반투자자 투자한도 완화 | 자본시장법 시행령 [대표 확정 필요 — 조항] | 1인당 한도 [대표 확정 필요]원 이내에서 일반투자자 모집 허용 |

### 3.4 투자자 보호 방안
