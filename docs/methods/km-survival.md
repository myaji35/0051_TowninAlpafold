# Kaplan-Meier 생존 분석

## 개요
점포 개업 → 폐업까지의 시간을 생존 시간(T)으로 정의하고, 시점별 누적 생존 확률 S(t)을 비모수 추정한다.

## 수식

S(t) = prod_{i: t_i <= t} (1 - d_i / n_i)

- d_i: 시점 t_i의 폐업 수
- n_i: 시점 t_i 직전 운영 중 점포 수

## CI 추정
Greenwood 공식 — log-log 변환 95% CI.

## 가정
- 비정보적 절단 (informative censoring 없음)
- 위험률(hazard) 시간 가변 가능 — Cox 모델과 달리 비례 위험 가정 없음
- 동질 코호트 — 본 시스템은 동/카테고리 매칭으로 코호트 통제

## 본 시스템 적용
유사 동(인구 구조 + 카페 밀도 ±15%) 5개를 코호트로 묶어 12개월 생존 곡선 추정. n ≥ 30 미만이면 결과 신뢰도 ↓ 표시.
